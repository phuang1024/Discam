Discam is a set of computer vision tools for automated filming of
Ultimate Frisbee games.

The flagship feature of Discam is automatic panning and zooming to
follow the action in a game.
This can be applied as a post processing step after filming, dynamically
cropping the video.
It can also be deployed on a PTZ camera live during the game, but
requires extra mechanical parts.

In development.
